# Reto Técnico Automatización

Reto técnico para las pruebas automatizadas de la pagina https://serenity.is/demo/

## El reto fue construido en

* Java 8
* Gradle 5.2

## Para compilar los test usar

Usando Gradle
```json
$ gradle clean test 
```
